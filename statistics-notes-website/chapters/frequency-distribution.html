5.1 Introduction

Frequency is the number of times a given value of an observation or event occurs in a dataset. A frequency distribution is a table that groups data into different classes and shows the number of observations that fall into each class.

It is a comprehensive method to organize raw data for both qualitative and quantitative variables. The table contains categories of data in one column and corresponding frequencies in another column.

---

5.2 Types of Frequency Distribution

1. Simple Frequency Distribution

a) Raw Series / Individual Series

Raw data are observations recorded in their original form without organization. Each observation occurs with frequency one.

Example:
Marks of students:
55, 73, 60, 41, 60, 61, 75, 73, 58, 80
---

b) Discrete Frequency Distribution

In discrete data, values are exact and distinct. The frequency of each value is counted using tally marks.

Children (x) Tally Frequency (f)
0|1
1|||3
2|||||5
3|||3
4||2
5|1
Total 15
---

c) Continuous Frequency Distribution

When the data range is large or measured continuously, values are grouped into class intervals.

Class Interval Tally Frequency
0 – 25||2
25 – 50|||3
50 – 75||||||7
75 – 100|||3
Total 15
---

Types of Continuous Class Intervals

1. Exclusive Method

In this method, the upper limit of one class becomes the lower limit of the next class. The upper limit is excluded from the class.

Example: 0–25, 25–50, 50–75, 75–100 ---

2. Inclusive Method

In this method both upper and lower limits belong to the same class.

Example: 0–25 26–50 51–75 76–100 ---

3. Open-End Classes

Open-end classes do not specify the lower limit of the first class or upper limit of the last class.

Example: - Below 20 - 20–40 - 40–60 - 60–80 - Above 80 ---

Difference Between Exclusive and Inclusive Methods

Exclusive Method Inclusive Method
Upper limit excluded Upper and lower limits included
Used for continuous data Used for discrete integer data
No gaps between classes Gaps between class limits
---

2. Relative Frequency Distribution

Relative frequency is the proportion of observations in a class compared with total observations.

Relative Frequency = Class Frequency / Total Frequency
Example: 2 / 15 = 0.133 ---

3. Percentage Frequency Distribution

Percentage frequency converts relative frequency into percentages.

Percentage Frequency = Relative Frequency × 100
Example: 2 / 15 × 100 = 13.33% ---

4. Cumulative Frequency Distribution

Cumulative frequency is the running total of frequencies.

Class Interval Frequency Cumulative Frequency
0–2522
25–5035
50–75712
75–100315
---

5. Cumulative Percentage Frequency

Cumulative percentage is obtained by adding percentage frequencies successively.

Example: 13.33 → 33.33 → 79.99 → 100% ---

6. Univariate Frequency Distribution

Frequency distributions involving only one variable are called univariate frequency distributions.

---

7. Bivariate and Multivariate Frequency Distribution

If data involve two variables simultaneously, it is called a bivariate distribution.

If data involve more than two variables, it is called a multivariate distribution.

---

5.3 Construction of Frequency Distribution

Discrete Distribution

Steps:
  • List all values of the variable.
  • Use tally marks to count occurrences.
  • Convert tally marks to frequencies.
---

Continuous Distribution

Important Terms

  • Class Interval: Range of values (50–75)
  • Class Limits: Lower (L) and Upper (U)
  • Range: Largest value − Smallest value
  • Midpoint: (L + U) / 2
  • Frequency: Number of observations in class
Range = Largest Value − Smallest Value
Midpoint = (L + U) / 2
---

Sturges Rule

Sturges rule determines the number of class intervals.

K = 1 + 3.322 log10(n)
Where: - n = number of observations - K = number of class intervals ---

Class Width

Class Width = Range / Number of Classes
---

Steps to Construct Continuous Frequency Distribution

  1. Determine number of class intervals using Sturges rule.
  2. Calculate class width.
  3. Choose starting lower class limit.
  4. Construct class intervals.
  5. Use tally marks to count observations.
  6. Compute frequencies.
