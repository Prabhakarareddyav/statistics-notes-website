12.1 Introduction

A random variable assigns numerical values to outcomes of a random experiment.

12.2 Random Variable

A variable whose value depends on outcomes of a random experiment.

Example: Two coins → S = {HH, HT, TH, TT}

X (number of heads) → {2,1,1,0}

Types

  • Discrete Random Variable: Countable values
  • Continuous Random Variable: Infinite values within range

12.3 Probability Distribution

A distribution that assigns probabilities to all values of a random variable.

P(X = x) ≥ 0
Σ P(X = x) = 1

Types

  • Discrete Distribution (PMF)
  • Continuous Distribution (PDF)

12.4 Discrete Probability Distributions

Bernoulli Distribution

P(X = x) = p^x (1-p)^(1-x), x = 0 or 1
  • Mean = p
  • Variance = pq

Binomial Distribution

P(X = x) = nCx p^x q^(n-x)
  • Mean = np
  • Variance = npq
  • Conditions: Fixed trials, independent, constant probability

Poisson Distribution

P(X = x) = (e^-λ λ^x) / x!
  • Mean = λ
  • Variance = λ
  • Used for rare events

12.5 Continuous Probability Distributions

Normal Distribution

f(x) = (1 / σ√2π) e^(-(x-μ)² / 2σ²)
  • Mean = Median = Mode
  • Bell-shaped curve
  • Symmetrical distribution

Properties

  • 68.26% data within ±1σ
  • 95.44% within ±2σ
  • 99.73% within ±3σ

Standard Normal Distribution

Z = (X - μ) / σ
  • Mean = 0
  • Standard deviation = 1

Applications of Normal Distribution

  • Central Limit Theorem
  • Statistical inference
  • Quality control
  • Hypothesis testing
