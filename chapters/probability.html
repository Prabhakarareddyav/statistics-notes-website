Chapter 11 - Probability

11.1 Introduction

Probability deals with uncertainty and chance. It measures how likely an event is to occur. Terms like "likely", "chance", and "possibly" are related to probability.

Definition

Probability is a numerical measure of uncertainty about the occurrence of an event.

11.2 Set Theory Basics

Set: Collection of well-defined objects.

  • Finite Set: Limited elements (e.g., vowels)
  • Infinite Set: Unlimited elements
  • Null Set (φ): No elements

Important Concepts

  • Subset: A ⊂ B
  • Equal Sets: A = B
  • Universal Set: S or U

11.3 Operations on Sets

Union

A ∪ B = elements in A or B

Intersection

A ∩ B = common elements

Complement

Ā = elements not in A

Difference

A - B = elements in A but not in B

11.4 Basic Probability Concepts

  • Experiment: Operation with outcomes
  • Random Experiment: Outcome uncertain
  • Trial: Single performance
  • Outcome: Result of experiment
  • Sample Space (S): All possible outcomes
  • Event: Subset of sample space

Example:

Coin toss → S = {H, T}

Die → S = {1,2,3,4,5,6}

11.5 Types of Events

  • Equally Likely Events
  • Mutually Exclusive Events
  • Exhaustive Events
  • Complementary Events
  • Independent & Dependent Events

11.6 Classical Probability

P(A) = Number of favorable outcomes / Total outcomes
  • 0 ≤ P(A) ≤ 1
  • P(A) + P(Ā) = 1

11.7 Types of Probability

  • Classical Probability
  • Empirical Probability
  • Axiomatic Probability

11.8 Permutation & Combination

Permutation

nPr = n! / (n - r)!

Combination

nCr = n! / [(n - r)! r!]

11.9 Theorems of Probability

Addition Theorem

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Multiplication Theorem

P(A ∩ B) = P(A) × P(B) (Independent events)
P(A ∩ B) = P(A) × P(B|A) (Dependent events)

11.10 Conditional Probability

P(B|A) = P(A ∩ B) / P(A)
P(A|B) = P(A ∩ B) / P(B)

Key Properties

  • Probability lies between 0 and 1
  • P(φ) = 0
  • P(S) = 1
  • Total probability = 1
