4.1 Meaning and Definition

A table is a systematic arrangement of data in rows and columns. Tabulation may be defined as the systematic arrangement of classified numerical data in rows and columns according to certain characteristics.

Tabulation expresses data in a concise and attractive form which can be easily understood and used for comparison. It enables an investigator to locate the desired information quickly.

Statistical tables make it possible to present a huge mass of data in a detailed and orderly form. It facilitates comparison and often reveals patterns in data that are otherwise not obvious.

Before tabulation, data are classified and then displayed under different columns and rows.

---

4.2 Difference Between Classification and Tabulation

Classification Tabulation
Process of grouping raw data according to characteristics. Logical arrangement of data into rows and columns.
First step in organizing data. Second step in presenting data.
Condenses mass of data to identify similarities and differences. Simplifies complex data for comparison.
---

4.3 Objectives / Advantages of Tabulation

  • Simplifies complex data for easy understanding.
  • Facilitates comparison of related facts.
  • Helps in computation of statistical measures like averages, dispersion and correlation.
  • Presents facts in minimum space and avoids repetition.
  • Useful for preparing graphs and diagrams.
---

4.4 Disadvantages of Tabulation

  • Requires skill and knowledge to arrange data properly.
  • Lack of description about the nature of data.
  • Important details may not be emphasized.
  • Data may be misinterpreted.
---

4.5 Characteristics of a Good Table

  • A table should match the objective of the statistical enquiry.
  • It should be simple and easy to understand.
  • Table size should fit the page properly.
  • Large figures should be rounded when necessary.
  • Units of measurement must be specified.
  • Rows and columns should follow logical order.
  • Totals should be shown clearly.
  • Footnotes and sources must be provided.
  • If data is too large, multiple tables should be used.
---

4.6 Parts of a Statistical Table

A statistical table consists of the following components:
  • Table Number
  • Title of the Table
  • Head Note
  • Captions (Column Headings)
  • Stubs (Row Headings)
  • Body
  • Footnotes
  • Source Note

Explanation

Table Number: Used for identification and reference.

Table Title: A clear description of the table contents.

Head Note: Explains units of measurement such as "in thousands".

Captions: Column headings that explain column data.

Stubs: Row headings describing row categories.

Body: Contains the actual numerical data.

Footnotes: Additional explanation for data items.

Source: Indicates the origin of data.

4.7 Types of Tabulation

Complex Table

---

2. Based on Purpose

General Purpose Tables

These tables provide detailed information for general reference. Example: Census tables.

Special Purpose Tables

These tables provide summarized information for specific analysis such as percentages, ratios and index numbers.

---

3. Based on Originality

Primary Tables

Contain original statistical data without modification. Example: Rainfall or production records.

Derived Tables

Contain data derived from original tables such as percentages, averages and ratios.
